## Description
匍匐前進のモーションパックです。
このパッケージには「陸上自衛隊 第一～第五匍匐風 匍匐前進」5種と「軍隊風 匍匐前進」の計6種が含まれています。

このアセットには、下記のアニメーションが含まれます。
標準的なHumanoidリギングモデル用
- anm_crawl01
- anm_crawl02
- anm_crawl03
- anm_crawl04_01
- anm_crawl04_02
- anm_crawl05
